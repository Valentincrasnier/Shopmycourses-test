# Shopmycourses-test

Project Test for shopmycourse by Valentin crasnier

## Prerequisites

*   Ruby 2.2.0
*   Rails 4
*   Bundler

## Installation

```bash
git clone https://github.com/Valentincrasnier/Shopmycourses-test.git
cd Shopmycourses-test
bundle
```

## Development & Production

`rails s`
